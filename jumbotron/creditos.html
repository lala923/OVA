Créditos

Esta página web fue realizada por la docente Laura Reinoso como parte del proyecto “Objeto Virtual de Aprendizaje para la enseñanza de las funciones lineales basada en la Enseñanza para la Comprensión” asesorada por Pablo Ochoa para optar por el título "Magister en enseñanza de las Ciencias Exactas y Naturales" de la Universidad Nacional de Colombia.

Para el desarrollo de la misma, se utilizaron los siguientes recursos:

  • Librería Boostrap
  • GitHub
  • Plantilla de libros interactivos de Juan Guillermo Ribera Berrío.
  • Video tutoriales de Juan Guillermo Ribera Berrío para el manejo de las plantillas de actividades de descartes y de los libros interactivos
  • Plantillas de objetos interactivos del proyecto Descartes
  • Software Hot Potatoes
  • Software Descartes
  • Software Geogebra
  • Librería Processing en Javascript
  • Software Scratch
